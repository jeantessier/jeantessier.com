Components

The packages are:

jt.awt
Layout manager for display a tree structure (horizontally).
jt.awt.event
MouseEventListener implementation for highlighting and hyperlinking graphical components.
jt.net
Utility classes for URL encoding and decoding, as well as encoding and decoding CGI parameter strings.
jt.sql
A connection pool class for use with JDBC.
jt.util
Utility classes for processing templaces and managing a distribution queue for producer / consumer problems.
jt.util.commandline
A command line parser.
jt.util.log
A logging library.
