I discovered [Pygments](https://pygments.org/), a tool in Python to do syntax
highlighting for hundreds of languages.  I can run it locally to produce images
or HTML that I can then include in my blog.  It also supports many styles, and
can generate the CSS for them.

I ran it against my [regex](#2023-06-28) and [data-driven testing](#2023-10-20)
blog entries.  It had trouble with "&bsol;" + "n" strings in Perl snippets, and
the generated HTML for Java regexes broke the blog completely.  It is more
likely the Markdown processor for the blog that takes issue with "&bsol;"
sequences.  The Java regex snippets have quite a few.  But other than that, it
works great.
